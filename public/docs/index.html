Introduction

This is an application to check Loan statuses of DFU customers

Base URL: localhost:8000

This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Authenticating requests

This API is not authenticated.

Endpoints

POST api/get-token

Example request:
curl --request POST \
    "localhost:8000/api/get-token" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"email\": \"email@email.com\",
    \"password\": \"Pass&LSD2\",
    \"token_name\": \"BankSystem\\n\\nIf you are able to Login everything is okay, you\'ll get a 200 OK response with the response:.\"
}"

API to view loan status

To access the API, a registered user must pass a valid token to the headers of the Loan API

Accept: application/json Authorization: Bearer {token}

Without the valid headers, the user will get a response:

Example request:
curl --request GET \
    --get "localhost:8000/api/get-loan-status" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"account_number\": \"5637294326\\n\\nIf you pass a ten-digit account number and everything is okay, you\'ll get a 200 OK response with the response:.\"
}"

const url = new URL(
    "localhost:8000/api/get-loan-status"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "account_number": "5637294326\n\nIf you pass a ten-digit account number and everything is okay, you'll get a 200 OK response with the response:."
};

fetch(url, {
    method: "GET",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());
